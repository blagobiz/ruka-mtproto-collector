Для каких запросов подходит

Страница и README заточены под запросы вроде «прокси для Telegram бесплатно», «MTProto proxy Telegram», «telegram proxy free» и «MTProto для Telegram».

Когда нужен VPN

MTProto открывает именно Telegram. Для браузера, YouTube, Instagram, ChatGPT и других приложений лучше использовать полноценный VPN.

Как использовать

  • Откройте mobile.html с телефона и нажмите на кнопку подключения.
  • Либо берите готовые файлы proxy_ru.txt, proxy_eu.txt и proxy_all.txt.
  • Если MTProto недостаточно, переходите в RukaVPN_bot и тестируйте VPN.
